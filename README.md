## 功能

1. 五笔输入(支持86版)
2. 五笔拼音混输，支持编码提示
3. 纯拼音输入
4. -/+/向上/向下键支持翻页
5. z键查询
6. 候选词横向/竖向排列支持
7. 候选词数量可配置
8. 候选窗口显示输入编码
